# Godot 教材仓库

这个仓库用于学习 Godot 4 的 2D 开发，包含一份可直接运行的小游戏脚本和三份配套教材文档。

## 适用人群
- 刚开始学 Godot 的同学
- 想系统补齐 2D 开发基础的人
- 希望通过小项目建立实战节奏的人

## 仓库结构
- `catch_game.gd`：接球小游戏核心脚本
- `assets/bg.png`：示例背景图
- `Godot全知识学习文档.md`：系统课程版教材
- `Godot脚本学习文档.md`：GDScript 专项教材
- `godot_小游戏记忆文档.md`：速记版操作与排错

## 快速开始
1. 安装 Godot 4.x（建议稳定版）。
2. 用 Godot 打开当前目录工程。
3. 新建一个 `Node2D` 作为根节点。
4. 将 `catch_game.gd` 挂到根节点。
5. 运行当前场景（`F6`）。
6. 使用左右方向键接球，`Enter` 在失败后重开。

## 推荐学习顺序
1. 先读 `godot_小游戏记忆文档.md`，确保你能跑起项目。
2. 再读 `Godot脚本学习文档.md`，把脚本基础补齐。
3. 最后按 `Godot全知识学习文档.md` 做阶段练习与验收。

## 学习目标（2~4 周）
- 能独立搭建 Godot 2D 场景结构
- 能编写与调试中小型 GDScript 逻辑
- 能完成一个可导出的小游戏 Demo（含 UI、音效、存档）

## 建议开发规范
- 每次只改一个功能点，再运行验证。
- 每次练习都写一条 commit 信息（如：`feat: add score ui`）。
- 出现 bug 先用 `print` 和断点定位，再改代码。

## Git 分支建议
- `master/main`：稳定版本
- `feature/*`：功能开发分支（当前你正在使用）
