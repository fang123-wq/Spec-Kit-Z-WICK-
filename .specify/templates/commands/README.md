# Command Templates

This directory contains minimal command templates used by the Spec-Kit flow.

## Included Commands

- `specify.md`: Create or update `spec.md` with constitutional alignment.
- `plan.md`: Produce `plan.md` and enforce Constitution Check gates.
- `tasks.md`: Generate dependency-ordered `tasks.md` with gate evidence tasks.
- `constitution.md`: Update constitution and synchronize dependent templates.

## Constitutional Gates (v2.0.0)

Every command template in this directory must preserve these checks:

1. Code quality gates are defined and measurable.
2. Testing standards are explicit and verifiable.
3. UX consistency criteria are explicit for user-facing changes.
4. Performance budgets and measurement methods are explicit.
